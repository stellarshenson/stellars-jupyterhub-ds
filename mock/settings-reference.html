Docker Proxy (limited-docker users)

VariableValueDescription
JUPYTERHUB_DOCKER_PROXY_SOCKET_DIR/var/run/jupyterhub-docker-proxy-socketsPath inside the hub container where the in-process proxy writes per-user listener sockets; backed by a named docker volume (no host path)
JUPYTERHUB_DOCKER_PROXY_USER_COMPOSE_PROJECT_TEMPLATE{username}_containersPython str.format template for per-user com.docker.compose.project label stamped on ad-hoc docker run containers when a docker-limited group enables it. Placeholders - {compose_project} (hub project), {username}. User's own docker compose -p is still respected when the group's allow-override is on
JUPYTERHUB_DOCKER_PROXY_LABEL_PREFIXjupyterhub.docker.proxyLabel namespace; proxy stamps <prefix>.owner and <prefix>.managed on its resources. All filtering keys off <prefix>.owner
